COME HO REALIZZATO L'APPLICAZIONE WEB

I linguaggi utilizzati per la scrittura del codice sono: 

1)HTML;
2)CSS; 
3)PHP; 
4)SQL

L'editor che ho usato per scrivere l'applicazione è NotePad ++ (facilmente scaricabile dal link : https://notepad-plus-plus.org/downloads/)

Per quanto riguarda il web server ed il DBMS, ho optato per l'utilizzo di XAMPP
un software multipattaforma che offre tutti gli strumenti necessari per l'utilizzo di PHP, tra cui appunto:

1) il relational DBMS MySQL,
2) il WebServer Apache.

XAMPP è completamente gratutito ed è facilmente scaricabile da questo link : https://www.apachefriends.org/it/index.html
